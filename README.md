## 🚀 Featured Project

### 🗂️ Athlantics

> *A full-stack platform that helps players to be recognized in society*

|Feature       |Details                            |
|--------------|-----------------------------------|
|**Frontend**  |Next.js + TypeScript               |
|**Backend**   |TypeScript + PostgreSQL            |
|**Auth**      |Role-Based Access Control (RBAC)   |
|**Real-time** |Server-Sent Events (SSE)           |
|**ORM**       |Prisma / Kysely (type-safe queries)|
|**Validation**|Zod schema validation              |

- 🎯 Streamlined complaint registration, tracking & resolution workflow
- 🔐 Role-based admin panel for ticket assignment & resolution
- ⚡ Real-time status updates via SSE
- 🛡️ Secure input validation with Zod
